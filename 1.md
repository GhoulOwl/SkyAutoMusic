# 原理说明
光遇通过键盘按键触发音符，该软件做的事情就是给电脑模拟一个虚拟键盘，然后使用曲谱自动按节奏敲击按键而已，在游戏中就是触发音符的效果，在记事本中就是打字的效果，因此不属于修改游戏数据的外挂而无法被监测，而更接近于辅助脚本

# 温馨提示
本人练习徒手弹背谱多小时后发现自己很菜，体会练习的艰辛，遂尝试自动化解决，试用其他基础版本后发现使用多有不便，于是加入更多的功能，用于体验游戏中不同的场景搭配不同乐器带来的舒适感觉，这并不影响个人和其他玩家体验友爱美好的游戏氛围。但值得注意的是，游戏中存在音乐玩家确实通过大量的努力徒手演奏令人惊艳，请使用该工具的光子们理解他们的不易并尊重其劳动成果，分享音乐的行为本身就很美好了，不妨在适当的时候把真相可以告诉聆听你的光子们，这没什么可耻的。

# 特别注意
针对歌曲有3种状态：播放中，暂停中，未播放。自动当前播放完成和主动点击“结束按钮”都可以切换状态未“未播放”，“点击播放按钮从头开始”或“退出软件”都需要当前歌曲处于“未播放”状态才能操作，否则会导致程序异常允许或无响应，若误操作则可任务管理器结束重新允许即可。
# 主要功能介绍
+ 支持任何符合格式的曲谱文件，文件存放于songs目录内
+ UI的页面支持文件名筛选文件
+ 支持快捷键F11播放和暂停
+ (todo)实时的手动速度调节
+ 辅助演奏：即只需要简单地按下鼠标的左右键即可自动触发一次弹奏的声音
+ (todo)节凑显示：搭配辅助演奏可以更好地按节奏敲击键盘
+ (todo)模拟的真实演奏：随机且自然的变速和失误
+ 进度条功能：指示实时进度，也可以拖动进度条控制位置
+ (todo)播放器功能：自动开始下一曲，随机开始下一曲
+ (todo)键盘映射多组配置：适配通用按键类音乐表演，不同的游戏的相同音符可能对应不同的按键组合
+ 调试模式：不实际按下按键仅有按键log输出
